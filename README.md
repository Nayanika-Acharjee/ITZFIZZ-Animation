

# 🚀 Scroll-Driven Hero Section Animation

This project demonstrates a refined approach to scroll-based interactions, combining performance-focused animations, structured layout, and modern design principles to deliver a seamless user experience.
### 🌐 *Live Demo*:
👉 https://nayanika-acharjee.github.io/ITZFIZZ-Animation/

### ✨ *Features*:

*🎬 Scroll-driven animation* 
- Bike movement synced with scroll  
- Smooth GSAP ScrollTrigger animations  

*🔤 Dynamic headline reveal*  
- "WELCOME ITZFIZZ" appears letter-by-letter  
- Reveal synced with banner expansion  

*🪟 Glassmorphism metric cards*  
- Subtle opacity + blur effect  
- Smooth staggered entry animation  

*🎯 Sticky scene design*  
- Fixed background SVG  
- Layered animation (road, bike, text)  

### 🛠️ *Tech Stack :*
- HTML5  
- CSS3  
- JavaScript  
- GSAP (Scroll-Trigger)
- SVG (Scalable Vector Graphics) 
- Transform & GPU Acceleration 
- Glassmorphism UI
### ⚙️ *How It Works :* 
- Scroll controls the animation timeline
- Banner expands horizontally based on scroll
- Letters appear progressively as the banner reveals
- Cards animate in during later scroll stages
- Layered elements create a depth/parallax effect
